# RRFS_plot_python

**This repository contains various Python scripts used for producing graphics of Rapid Refresh Forecast System (RRFS) grib2 output at NOAA's Environmental Modeling Center (EMC).  A description of each script is provided below.**

<ins>plot_rrfs_9members.py</ins>: Generates postage stamp plots of an RRFS 9-member ensemble for 2-m temperature, 2-m dew point temperature, composite reflectivity, max/min hourly 2-5 km updraft helicity, most unstable CAPE/CIN, and max hourly upward vertical velocity.
Command line arguments: Cycle date/time (YYYYMMDDHH format), forecast hour
